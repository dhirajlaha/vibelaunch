# Image Asset Notes

This folder stores static image assets for the CRM logistics website.

## Optimization Guidance
- Prefer WebP for photos/illustrations.
- Keep standard assets under 200KB where possible.
- Ensure meaningful images include descriptive alt text in consuming components.

## Planned Assets
- hero-dashboard.webp: landing hero visual, 1440x900, under 300KB
- service-ops.webp: services section visual, 1200x800, under 200KB
- about-team.webp: about section visual, 1200x800, under 200KB
