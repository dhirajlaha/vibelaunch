# Performance Validation Report

## Target
- Primary pages usable within 3 seconds under standard mobile network conditions.

## Measurement Method
- Use Lighthouse performance audit for `/`, `/services`, `/about`, `/faq`.
- Record First Contentful Paint, Largest Contentful Paint, and Total Blocking Time.

## Thresholds
- FCP <= 1.8s
- LCP <= 2.5s
- TBT <= 200ms
