# Landing Page Manual Validation Checklist

- [ ] Value proposition visible above the fold at 360px, 768px, 1280px
- [ ] Feature highlights load without JavaScript errors
- [ ] Navigation links from landing page are functional
- [ ] Page title is unique and descriptive
- [ ] Meta description is present in head
- [ ] CTA buttons are keyboard-accessible with visible focus ring
- [ ] Color contrast passes for headline, body, and CTA
- [ ] Page renders correctly in latest Chrome and Firefox
- [ ] Static build completes successfully
